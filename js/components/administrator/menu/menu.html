Helpful Tip

Note: Need to add a new user? In addition to the basic roles such as GM, Bar Manager, Kitchen Manager, etc. Artecsan also offers an “Auditor” role. This role allows a user to count your inventory without having access to any other features. For example, you may want a bartender to count inventory without giving them access to sensitive data. You can accomplish this via the auditor's role.
